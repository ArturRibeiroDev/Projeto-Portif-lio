## 🚀 Tecnologias

| Tecnologia | Função |
|---|---|
| **HTML5** | Estrutura semântica |
| **CSS3** | Estilização modular |

## ✨ Funcionalidades

- ✅ Seção hero com apresentação
- ✅ Grid de projetos
- ✅ Header com navegação
- ✅ CSS modular (section, header, utility)

## 🏗️ Estrutura

```
/styles
├── global.css, index.css, section.css
├── header.css, utility.css
```

## 🚦 Como Rodar

Abra `index.html` no navegador.

## 💡 O que foi aprendido

- Estruturação de um portfólio do zero
- Organização do CSS em arquivos por seção
- Primeiro contato com layout responsivo
